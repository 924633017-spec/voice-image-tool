# 图片音频字幕互动工具 — 实施计划

> 2026-06-18

## 总览

共 6 个阶段，按优先级递进。先跑通核心链路（图片+录音+字幕+分享），再补用户体系和付费。

---

## Phase 1：项目脚手架 + 数据库

**目标**：跑起来，能登录

- [ ] `npx create-next-app@latest` 初始化项目
- [ ] 配置 shadcn/ui + Tailwind
- [ ] 配置 Prisma + PostgreSQL 连接
- [ ] 创建数据模型迁移（User, Project, Hotspot, AudioClip, SubtitleSegment）
- [ ] 配置阿里云 OSS SDK
- [ ] NextAuth.js 邮箱登录
- [ ] 基础布局（导航栏 + 侧边栏壳子）

**产出**：能注册登录的空项目，数据库就绪

---

## Phase 2：编辑器核心

**目标**：能在图片上放热点、录音、看字幕

- [ ] 图片上传组件（拖拽/选择 → 上传 OSS → 显示在画布）
- [ ] 图片画布（支持缩放、拖拽）
- [ ] 热点标注（点击放置标记、拖拽移动、删除）
- [ ] 录音面板（MediaRecorder → 上传 OSS → 关联到热点）
- [ ] AI 转写集成（录音完成 → 调用阿里云语音识别 → 存入 SubtitleSegment）
- [ ] 字幕编辑器（文本修正、时间轴拖拽调整）
- [ ] 项目保存/加载

**产出**：核心编辑功能可用

---

## Phase 3：播放页 + 分享

**目标**：生成可分享的 H5 页面

- [ ] 播放页 SSR 渲染（读项目数据，渲染图片+热点）
- [ ] 热点交互（点击播放音频、暂停、切换热点）
- [ ] 字幕同步（ontimeupdate 驱动逐句高亮）
- [ ] 字幕显示开关
- [ ] 分享链接生成 + 二维码
- [ ] 可选密码保护
- [ ] 移动端适配

**产出**：做完一张图，分享出去别人能看能听

---

## Phase 4：工作台 + 官网

**目标**：用户能用起来

- [ ] 工作台项目列表（卡片视图、搜索、筛选）
- [ ] 项目操作（新建、编辑、删除、分享、查看数据）
- [ ] 官网首页（产品介绍 + 定价 + 案例）
- [ ] 免费版限制逻辑（3 项目、10 热点、2 分钟录音、水印）

**产出**：完整产品体验闭环

---

## Phase 5：付费 + 用户体系

**目标**：开始收钱

- [ ] 微信支付 / 支付宝集成
- [ ] 订阅管理（升级/降级/续费/取消）
- [ ] 用量统计（项目数、录音时长）
- [ ] 企业版功能（团队管理、成员邀请、白标）
- [ ] 数据看板（播放量、热点点击排行）

**产出**：商业化就绪

---

## Phase 6：部署上线

**目标**：生产环境运行

- [ ] 阿里云 ECS 部署（Docker + Nginx + SSL）
- [ ] CI/CD（GitHub Actions）
- [ ] 监控 + 日志（Sentry）
- [ ] 域名 + 备案

**产出**：正式上线

---

## 技术栈

| 层 | 选型 |
|----|------|
| 框架 | Next.js 14 App Router |
| UI | shadcn/ui + Tailwind CSS |
| 数据库 | PostgreSQL + Prisma |
| 存储 | 阿里云 OSS |
| 认证 | NextAuth.js |
| 语音识别 | 阿里云语音识别 API |
| 支付 | 微信支付 / 支付宝 |
| 部署 | 阿里云 ECS + Docker |
