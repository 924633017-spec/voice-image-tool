# 图片音频字幕互动工具 — 产品设计文档

> 2026-06-18 | Status: Draft

## 一、产品定位

在图片上添加可点击的热点标记，每个标记绑定一段录音和同步字幕。制作端完成标注后，生成 H5 分享链接，观众点击标记即可听音频、看字幕。

**目标用户**：文旅从业者（博物馆、景区、展览公司）、教育者、自媒体创作者。

**差异化**：国内唯一面向创作者的「图片热点 + 录音 + 同步字幕」SaaS 工具。ThingLink 无中文版，国内 AI 讲解工具是消费端（拍照识别），不是创作端。

## 二、商业模式

C 端免费引流 + B 端订阅付费。

| 层级 | 价格 | 核心差异 |
|------|------|---------|
| 免费版 | ¥0 | 3 项目、10 热点/图、2 分钟/录音、品牌水印 |
| 专业版 | ¥29/月 | 无限项目、50 热点/图、10 分钟/录音、无水印 |
| 企业版 | ¥299/月 | 团队协作、白标、API、数据看板 |

## 三、技术架构

```
制作端 (SPA)         播放端 (SSR)          API Routes
/editor/[id]         /play/[id]            /api/*
图片标注+录音+字幕     公开分享+热点交互     认证/存储/AI/支付

PostgreSQL + Prisma    阿里云 OSS          阿里云语音识别
用户/项目/热点/订阅      图片/音频存储        录音 → 带时间戳字幕
```

**选型**：Next.js 14 App Router + shadcn/ui + Tailwind CSS + PostgreSQL + Prisma

## 六、数据模型

```
Project          Hotspot           AudioClip        SubtitleSegment
─────────       ──────────        ──────────       ────────────────
id              id                id               id
title           projectId ──▶     hotspotId ──▶    audioClipId
coverImage      x, y (%)          audioUrl         text
userId          color             duration         startTime
teamId          icon              transcript       endTime
settings        title
isPublic        order
createdAt
```

- 热点坐标用百分比，适配不同屏幕
- 一个热点只有一段音频
- 字幕片段带时间戳，支持逐句高亮

## 七、核心流程

### 制作流程
上传图片 → 放置热点标记 → 点击标记 → 打开录音面板 → 在线录音 → AI 转写字幕 → 编辑修正字幕 → 保存 → 发布生成链接

### 播放流程
打开分享链接 → 看到图片 + 热点标记 → 点击标记 → 音频播放 + 字幕逐句同步高亮 → 可切换字幕显示

## 八、技术细节

### 录音
- 浏览器 MediaRecorder API（opus/webm 编码）
- 上传至阿里云 OSS，存为 MP3

### AI 转写
- 录音完成后调用阿里云语音识别 API
- 返回带时间戳的文本，自动切分为 SubtitleSegment

### 字幕同步
- 播放时 AudioElement.ontimeupdate 驱动
- 根据 currentTime 匹配对应字幕片段
- 当前字幕高亮 + 自动滚动

### 分享页
- SSR 渲染，微信/Facebook 分享预览友好
- 生成二维码，支持密码保护
- 热点点击数据埋点（企业版）
